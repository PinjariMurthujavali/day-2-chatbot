\# Day 2: AI Chatbot with Memory 



AI chatbot that remembers conversation history.



\## Features

\- Continuous conversation mode

\- AI remembers past messages

\- Full conversation context maintained

\- Exit gracefully with 'exit' command



\## Setup

```bash

python -m venv venv

venv\\Scripts\\activate

pip install groq python-dotenv

```



\## Usage

```bash

python chatbot.py

```



Ask multiple questions - AI remembers previous messages!
