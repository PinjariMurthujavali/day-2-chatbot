## 📌 About This Project

Part of my **AI 90-Day Challenge** — building one real AI project every single day.

Day 2 upgrades the chatbot with **conversation memory** — the AI now remembers everything said earlier in the chat, making conversations feel natural and continuous instead of one-off Q&A.

## ✨ Features

- 🔁 **Continuous Conversation Mode** — chat as long as you want, no restarts needed
- 🧠 **AI Remembers Past Messages** — context carries forward across the whole session
- 📝 **Full Conversation Context Maintained** — every message builds on the last
- 🚪 **Graceful Exit** — type `exit` to end the chat anytime

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **AI/LLM:** Groq API (Llama 3.3)
- **Libraries:** `python-dotenv`

## 🚀 How to Run

```bash
# Clone the repo
git clone https://github.com/PinjariMurthujavali/day-2-chatbot.git
cd day-2-chatbot

# Setup virtual environment
python -m venv venv
venv\Scripts\activate

# Install dependencies
pip install groq python-dotenv

# Add your Groq API key in a .env file
GROQ_API_KEY=your_key_here

# Run the chatbot
python chatbot.py
```

## 💬 Usage

Ask multiple questions in a row — the AI remembers everything you've said before and responds with full context, just like a real conversation.
